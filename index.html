🌌 Stellar Classification using Machine Learning

This project focuses on classifying Stars, Galaxies, and Quasars (QSOs) using Machine Learning techniques. We applied both Support Vector Machine (SVM) for supervised classification and KMeans for unsupervised clustering on data from the Sloan Digital Sky Survey (SDSS).
